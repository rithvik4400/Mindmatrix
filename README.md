# Android Development Projects Collection

Welcome to my Android development repository!
This repo contains multiple apps and codelabs built while learning **Kotlin** and **Jetpack Compose**.

---

##  Projects Included

###  Core Apps

* 🍰 **Dessert Clicker App**
  Tracks dessert sales and revenue dynamically.

* 🧁 **Cupcake App**
  Order cupcakes with quantity selection and summary screen.

* 🌍 **Mars App**
  Displays Mars data using API integration.

* 🏁 **Race Tracker App**
  Simulates race progress with timers and UI updates.

* 💬 **Reply App**
  Email-style UI with list and detail screens.
